# YAML Schema

YAML Schema is a specification to define the structure and constraints of your YAML documents, in YAML.

The YAML Schema specification is based on JSON Schema ([https://json-schema.org/](https://json-schema.org/)), but expressed as YAML.

## Getting Started

If you're new to YAML Schema, we recommend starting with these guides:

1. **[Installation](getting-started/installation.md)** - Install the `ys` command-line tool
2. **[Basic Usage](getting-started/basic-usage.md)** - Learn the fundamentals of schema validation
